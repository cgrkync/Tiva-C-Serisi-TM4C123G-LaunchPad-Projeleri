# Tiva C Serisi LaunchPad ile 16x2 LCD Arayüzü
Kodlar Code Composer Studio'da yazılmış ve Tiva C Serisi TM4C123G LaunchPad üzerinde test edilmiştir.

## Gerekli Malzemeler
- 16x2 LCD Display
- 10k Potansiyometre
- Breadboard

## Bağlantılar
| LCD      | TM4C123G LaunchPad       |
|----------|--------------------------|
| VSS      | GND                      |
| VDD      | VBUS                     |
| V0       | Potentiometer            |
| RS       | PB0                      |
| RW       | GND                      |
| E        | PB1                      |
| D4       | PB4                      |
| D5       | PB5                      |
| D6       | PB6                      |
| D7       | PB7                      |
| A        | 5V                       |
| K        | GND                      |
